## Hinweis: Raw-Text vs. GitHub Pages

- **`raw.githubusercontent.com`** → immer `Content-Type: text/plain`, direkt maschinenlesbar, ideal für Claude Projekt-Instruktionen
- **GitHub Pages** → rendert `.md` als HTML; nützlich für menschliche Ansicht

> Für Claude Projekt-Instruktionen ausschliesslich die Raw-URLs verwenden.
